# kawat-html

HTML tree cleaning and tag normalization for the kawat web content extraction library.

## Features

- **Tree cleaning**: Remove unwanted elements (scripts, styles, navigation, etc.)
- **Tag normalization**: Convert semantic HTML5 tags to consistent output format
- **Link density analysis**: Identify and remove high-link-density boilerplate
- **Image preservation**: Optional image element handling
