# kawat-readability

Readability-based content extraction fallback for the kawat web content extraction library.

Wraps the `dom_smoothie` crate to provide the readability algorithm fallback used in trafilatura.

## Features

- **Readability algorithm**: Mozilla's readability algorithm implementation
- **Fallback extraction**: Used when primary extraction methods produce short content
- **Configurable parameters**: Adjustable text length thresholds
