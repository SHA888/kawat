# kawat-output

Output format conversion for the kawat web content extraction library.

Converts extracted content to multiple formats: TXT, Markdown, JSON, XML, XML-TEI, CSV, HTML.

## Features

- **Multiple output formats**: TXT, Markdown, JSON, XML, XML-TEI, CSV, HTML
- **Metadata inclusion**: Optional metadata in structured formats
- **Streaming support**: Efficient conversion without loading entire document into memory
