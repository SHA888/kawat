# kawat-metadata

Metadata extraction for the kawat web content extraction library.

Extracts structured metadata from HTML documents including title, author, date, categories, tags, and license.

## Features

- **Title extraction**: Multiple fallback strategies for page title
- **Author extraction**: Byline, JSON-LD, and meta tag parsing
- **Date extraction**: Via htmldate-rs with configurable date ranges
- **Open Graph support**: og:* meta tag parsing
- **JSON-LD parsing**: Schema.org structured data extraction
- **Categories & tags**: Custom classification extraction
